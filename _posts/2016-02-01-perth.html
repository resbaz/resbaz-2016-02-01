Below are the tools on offer at Perth ResBaz 2016. Registration will cost $50 (excl. GST).

The registration fee includes morning tea, lunch, and afternoon tea, along with refreshments and drinks at the end of the first day, as well as dinner and drinks on the second day. classes: - class: Main Streams tools: - name: Using the Unix shell id: unix blurb: Use of the shell is fundamental to using a wide range of other powerful tools and computing resources (including “high-performance computing” supercomputers). These lessons will start you on a path towards using these resources effectively. - name: Programming with R id: r1 blurb: R is widely used across a range of disciplines for statistical computing and generating publication-quality figures. In this Software Carpentry workshop, you’ll learn the fundamentals of R for data science, Unix Shell for interacting with other programs, as well as version control and useful concepts for scientific computing such as breaking down problems into modular units, reproducibility, tracking changes to code, collaborating with others, and working with structured data. - name: Programming with Python id: python blurb: Python is a powerful, general purpose programming language that has several popular scientific libraries that make it ideal for research. This workshop will equip you with the essential skills you need to use the Python programming language. It covers fundamentals as well as creating reusable functions, graphing with matplotlib and what to do to find and fix errors in your code. - name: Version Control with Git id: git blurb: Version control is the lab notebook of the digital world - it’s what professionals use to keep track of what they’ve done and to collaborate with other people. Keeping a record of what was changed, when, and why is extremely useful for all researchers if they ever need to come back to the project later on. - class: Minor Electives tools: - name: Document editing with LaTeX id: latex blurb: Learn how to make your technical or scientific documents look brilliant with LaTeX typesetting. - name: Visualizing your data on the web using D3 id: d3_js blurb: D3 is a JavaScript library for working with data and displaying it online. D3 helps you bring data to life using HTML, SVG, and CSS. This workshop will introduce you to the basics of using D3 to create beautiful, animated and interactive graphics out of your data. - name: Natural language processing with NLTK id: nltk blurb: Researchers are often confronted with natural language datasets so large that not even teams of researchers would be able to analyse by hand. (Even just reading through the 100 million word British National Corpus, for example, would take around 10,000 hours.) Python's Natural Language Toolkit (NLTK) provides a powerful means of investigating large bodies of text for patterns that may otherwise be very difficult to find. In this workshop, you’ll learn key concepts and practices in Corpus Linguistics/Distant Reading, investigate example datasets and strategies for implementing NLTK within your own research. - name: Analysing qualitative data with Nvivo id: nvivo blurb: schedule: Each day at ResBaz will be split into daytime and evening events. The day will be dedicated to classroom teaching time where we will fill your brains with the latest digital research skills. The evening will be about networking with your fellow cross-disciplinary researchers and having fun.

During the day classes are broken into three 90 minute sessions, starting at 10.30 am, 1.30 pm and 3 pm. Morning or afternoon tea will be served between the sessions.
